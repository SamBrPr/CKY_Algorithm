# CKY Algorithm

This Program analyzes whether a word is part of a certain grammar utilizing the Cocke–Younger–Kasami algorithm. Aditionally, this program can also convert any Context-Free Grammar into Chomsky Normal Form, as well as working with probabilistic grammars.
